# gd32-tools-xpack-arm-none-eabi-gcc
This repository is created for xPack GNU Arm Embedded GCC to be added as submodule to gd32-cmake repositories.

**xPack GNU Arm Embedded GCC Toolchain** version is **_xpack-arm-none-eabi-gcc-11.3.1-1.1_**.
   - You can use following link to download other versions:
   		- https://github.com/xpack-dev-tools/arm-none-eabi-gcc-xpack/releases
		
# xPack GNU Arm Embedded GCC

The **xPack GNU Arm Embedded GCC** (formerly GNU MCU Eclipse Arm GCC)
is the **xPack** version of the **GNU Arm Embedded Toolchain**
maintained by Arm.

For more details, please read the corresponding release pages:

- <https://xpack.github.io/arm-none-eabi-gcc/releases/>
- <https://developer.arm.com/tools-and-software/open-source-software/developer-tools/gnu-toolchain>
